# AI Agent Guide & SOP

This is the standard operating procedure (SOP) for all AI agents working in this repository.

## Step-by-Step Workflow

**Step 1: Ingestion & Alignment**
- Read `context.md` to understand the project.
- Read `memory.md` to check what was recently changed.

**Step 2: Planning**
- For complex tasks, outline a plan before executing. Ensure the plan aligns with the project goals.

**Step 3: Execution**
- Write clean, documented, and testable code.
- Ensure any UI/UX changes align with the required design aesthetics (e.g., modern, dynamic, responsive).

**Step 4: Documentation (CRITICAL)**
- Update `memory.md` with a summary of the changes you just made. 
- Update `context.md` if your changes shift the architecture or complete an active task.
- If you ran into issues or left tasks unfinished, document them in `claude.md` or `memory.md` for the next agent.

By following this process, we ensure a seamless pair-programming experience regardless of which AI assistant is currently active.
